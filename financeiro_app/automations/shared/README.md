# Automações globais (`shared/`)

Scripts nesta pasta podem ser usados por **todos os setores**.

## Cadastro

1. Coloque o `.py` aqui (ex.: `meu_utilitario.py`).
2. Admin cadastra no sistema com:
   - `visibility`: `global`
   - `sector`: `shared`
   - `script_path`: `automations/shared/meu_utilitario.py`

## Exemplo existente

- `acessar_drive.py` — utilitário de exemplo registrado automaticamente no startup.
